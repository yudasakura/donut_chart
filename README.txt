INTRODUCTION
------------

The Donut Chart module adds a configurable formatter for numeric fields that 
allows the display to be a stylish donut chart.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/donut_chart

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/donut_chart


REQUIREMENTS
------------

This module requires only one module to work:
 * Field (included on core)


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.


CONFIGURATION
-------------

 * Configure the display of the donut chart (colors and widths) in 
   /admin/structure/type/manage/<entity_name>/display
